In order to run this build, you can follow these steps:

$brew install jruby	" make sure you have installed the jruby, and you should add jruby to PATH"
$jgem install cuke4duke 

After install cucke4duke, you should make sure it is installed in jruby. check it with:

$jgem list

check you have installed the cuke4duke,

cuke4duke (0.4.4)

and if the version is 0.4.4, make sure you install the cucumber which version is

cucumber (1.1.2)
jgem install cucumber --version=1.1.2

After instal cuke4duke and cucumber, at first you should start a device or a emulator,
then you can follow the command.txt to run the test.
